linux基础

Date: April 12, 2020  10:29:44 Category: linux

文件目录类等

  • pwd：当前目录
  • ls：当前目录下的文件
    • ls -al
  • cd：切换目录
    • cd ~：回到用户home目录
  • mkdir：创建文件夹
    • mkdir -p：多级目录一次性创建
  • rmdiv：删掉空目录
  • rm -rf：删非空目录
  • touch：创建空文件
  • cp：拷贝
    • cp 文件 目录：拷贝该文件到这个目录下
    • cp -r 文件夹 目录：拷贝整个文件夹到这个目录下
    • \cp：强制覆盖
  • mv
    • mv oldname newname：重命名
    • mv 文件 目录：移动文件到目录下
  • cat：浏览文件
    • cat -n：显示行号
    • cat|more：分页显示
  • less：类more，适合大文件
  • head：默认显示文件开头前10行
    • head -n 5：显示前5行
  • tail：类head，显示文件后10行
    • tail -f 文件：实时监控文件的所有更新
  • >：输出重定向，把输出的内容覆盖写入某文件
  • >>：把输出的内容跟追加到某文件
  • cal：当前日历信息
  • ln：软链接（快捷方式）
    • ln -s 原文件 链接文件
    • rm -rf 软链接：删除软链接
  • echo：字符串输出
    • echo “字符串”：输出字符串
    • echo $PAHT：输出环境变量
  • history：查看历史执行指令（含编号）
  • ！历史执行指令编号：执行该历史指令

用户和用户组管理

  • useradd：添加新用户
  • userdel：删除用户
  • usermod：修改账号
  • passwd：用户密码
  • groupadd：添加用户组
  • groupdel：删除用户组
  • groupmod：修改用户组

vi/vim

文本编辑器

  • 正常模式：刚进入的模式
    • i：进入输入模式
    • ：：进入命令行模式
    • 快捷键
      • gg：首行
      • G：末行
      • dd：删除
      • /关键词：查找
      • yy：复制当前行
        • 5yy：复制当前及下面5行
      • p：粘贴
    • :set nu：显示行号
    • :set nonu：不显示行号
  • 输入模式
    • 键盘esc：回到正常模式
  • 命令行模式
    • :wq：保存并退出
    • :q：退出
    • :q!：强制退出
    • 键盘esc：回到正常模式
